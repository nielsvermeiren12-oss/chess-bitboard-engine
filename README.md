# Chess Game

I decided to create my own Chess bitboard engine with a chess computer to test
your skills against.

## Features

Bitboard chess engine: 
- Chess moves
- Capturing
- Check and checkmate
- Castling
- Promotion
- Stalemate
- En passant

Chess computer:

- Chess AI using MINIMAX search tree
- Alpha beta pruning
- Quiescence search
- Position lookup tables in evaluation function
- Depth of 5-6
